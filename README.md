# web_project_portfolio_es

Proyecto de portafolio personal - Laura Aldana

Descripción del proyecto
Este es un portfolio web personal que presenta mis habilidades como desarrolladora Full-Stack Junior. Incluye una sección de presentación, tecnologías, proyectos destacados y contacto.

El objetivo de este proyecto es mostrar mis conocimientos en desarrollo web y mi capacidad para crear interfaces modernas, limpias y responsivas.

Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, diseño responsivo)
- Git y GitHub

Técnicas utilizadas

- Maquetación con Flexbox
- Uso de pseudo-elementos (::before)
- Diseño visual con gradientes y efectos
- Organización semántica del HTML
- Control de layout y alineación de componentes
